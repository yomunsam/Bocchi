# Src/CloudServer/

Reserved for the Cloud Server component (see `Docs/Architecture.md §2.3` and milestone M7).

The Cloud Server hosts the small subset of features that must run online — comments, dynamic note publishing, Webmention, subscription callbacks, lightweight analytics, authenticated dynamic APIs.

In M1 ~ M6 this directory contains no projects. M7 will add an ADR (architecture decision record) here and only then begin implementation.
